Bug Bunny is a local-first proof console for authorized web research. It works with local labs, systems you own, and explicitly authorized in-scope targets, including programs hosted on HackerOne, Bugcrowd, or Intigriti. This recording stays local, using two synthetic accounts, controlled objects, and a four-request ceiling.

In Proof Lab, I paste each account's DevTools cURL capture. Tokens, object IDs, URLs, markers, and bodies stay ephemeral: never rendered or saved.

Before traffic, the preview confirms one endpoint shape, one changed locator, and two distinct sessions. I add harmless owner-only markers and confirm control.

The matrix runs B-to-B, A-to-A, then derives B-to-A by changing only the session. A's marker appears under B, so Bug Bunny stops immediately at three of four requests.

The write-once receipt keeps hashes, marker booleans, timings, network controls, and integrity metadata. An optional, model-configurable AI reviewer can challenge the redacted evidence, but it never controls the deterministic verdict. Human review is still required.
